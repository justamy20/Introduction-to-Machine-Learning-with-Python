# Introduction-to-Machine-Learning-with-Python

Repository ini berisi hasil pembelajaran dan reproduksi kode dari buku *Introduction to Machine Learning with Python* karya Andreas C. Müller dan Sarah Guido.

 Tujuan

Tujuan dari repository ini adalah untuk:

* Memahami konsep dasar Machine Learning
* Mencoba langsung implementasi algoritma menggunakan Python
* Menjelaskan teori di balik model yang digunakan

Ringkasan Chapter

Chapter 1 - Introduction

Pada chapter ini dibahas pengenalan Machine Learning, mulai dari apa itu Machine Learning, jenis-jenisnya, serta alur dasar dalam membangun model.

Chapter 2 - Supervised Learning

Chapter ini membahas metode supervised learning, yaitu model yang dilatih menggunakan data berlabel. Beberapa algoritma yang digunakan antara lain KNN dan Logistic Regression.

Chapter 3 - Unsupervised Learning

Di chapter ini dijelaskan bagaimana model dapat menemukan pola dalam data tanpa label, salah satunya menggunakan metode clustering seperti K-Means.

4 - Feature Engineering

Chapter ini membahas pentingnya mengolah data sebelum digunakan dalam model, seperti melakukan scaling dan transformasi data agar hasil model lebih optimal.

Chapter 5 - Model Evaluation

Pada bagian ini dijelaskan bagaimana cara mengevaluasi model agar kita tahu seberapa baik performanya, misalnya menggunakan cross-validation.

6 - Pipelines

Chapter ini membahas penggunaan pipeline untuk menggabungkan proses preprocessing dan model dalam satu alur yang lebih rapi.

Chapter 7 - Text Data

Di chapter terakhir ini dijelaskan bagaimana mengolah data teks menjadi bentuk numerik agar bisa diproses oleh model Machine Learning.

Tools yang Digunakan

* Python
* scikit-learn
* NumPy
* pandas
* matplotlib

Kesimpulan

Dari proses ini saya jadi lebih memahami bahwa Machine Learning tidak hanya soal membuat model, tapi juga bagaimana menyiapkan data, memilih metode yang tepat, dan mengevaluasi hasilnya dengan baik.
